# 快速开始指南

## 第一步：环境准备

### 1. 检查Python版本

```bash
python --version
```

确保版本是 Python 3.8 或更高。

### 2. 安装依赖

在项目根目录运行：

```bash
pip install -r requirements.txt
```

## 第二步：运行程序

### 启动应用

```bash
python run.py
```

### 首次使用

1. 程序启动后会弹出对话框要求输入用户名
2. 输入你的用户名（如：张三）
3. 点击确定

## 第三步：基本操作

### 1. 发现其他用户

- 程序启动后会自动发送广播发现其他用户
- 如果没有发现，点击**"🔄 刷新成员列表"**按钮
- 等待几秒，在线成员会出现在左侧成员列表中

### 2. 发送一对一消息

1. 在左侧成员列表中点击选择一个成员
2. 在底部输入框输入消息
3. 点击**"发送"**按钮或按回车键

### 3. 发送广播消息

1. 在底部输入框输入消息
2. 点击**"广播"**按钮
3. 消息将发送给所有在线成员

### 4. 发送文件

1. 在左侧成员列表中选择一个成员
2. 点击**"发送文件"**按钮
3. 选择要发送的文件
4. 等待对方接受
5. 查看传输进度

### 5. 接收文件

1. 当有人向你发送文件时，会弹出确认对话框
2. 点击"是"接受文件，或"否"拒绝
3. 接受的文件保存在项目根目录的`downloads/`文件夹中

## 常见问题

### Q1: 发现不了其他用户？

**解决方法**：
1. 确保所有用户在同一个局域网内
2. 检查防火墙设置，允许程序访问网络
3. 确认UDP端口8888未被占用
4. 点击刷新按钮重新搜索

### Q2: 消息发送失败？

**解决方法**：
1. 确认对方仍在线
2. 检查网络连接
3. 重新刷新成员列表

### Q3: 文件传输失败？

**解决方法**：
1. 确认TCP端口8889未被占用
2. 检查文件大小（不超过100MB）
3. 确保有足够的磁盘空间
4. 检查对方是否接受了文件传输

### Q4: 程序无法启动？

**解决方法**：
1. 检查Python版本（≥3.8）
2. 确认已安装PyQt6：`pip install PyQt6`
3. 检查是否有端口冲突

## 开发测试

### 在同一台电脑测试

1. 打开多个终端窗口
2. 在每个窗口中运行：`python run.py`
3. 输入不同的用户名
4. 测试消息和文件传输功能

### 局域网测试

1. 在不同的电脑上运行程序
2. 确保所有电脑连接到同一个WiFi或局域网
3. 测试各项功能
