1. 输入消息，点击"广播"
2. 所有实例都应该收到这条消息

## 💡 使用技巧

1. **多开测试**：可以在同一台电脑开多个实例测试

4. **端口占用**：默认使用UDP 8888端口

## 🐛 故障排除

### 问题：找不到其他用户
```
解决方案：
1. 确保在同一局域网
2. 检查防火墙设置
3. 点击"刷新成员列表"
4. 查看控制台日志
```

### 问题：消息发送失败
```
解决方案：
1. 确认对方在线
2. 确认已选中接收者
3. 重新启动程序
```

### 问题：程序无法启动
```
解决方案：
1. 检查Python版本: python --version
2. 安装依赖: pip install -r requirements.txt
3. 查看错误信息
```
